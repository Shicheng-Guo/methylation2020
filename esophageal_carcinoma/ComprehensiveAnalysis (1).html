3.1 Part 3.1: Constructing the methods for merging the CpGsites in a gene into a variable.

3.1.1 Summary: Because we have several CpGsites for each gene, but we want to get a overall estimate of each gene for constructing the panel. So I designed three methods for it. Firstly, I did the correlation test for all the CpGsites in each gene, and I removed all the CpGsites that showed little correlation with the others, which means that these CpGsites are not consistent with the overall methylation status in this gene. And after the removal of the undesired CpGsites in each gene, I constructed three methods to merge the CpGsites to a variable to represent the methylation status of each gene. The first is based on the PCA method, which I first do the PCA analysis in each of the gene, and get the PC1 variable to represent the overall methylation status of each gene. And I also generated the percent of explained variance of the PC1 for each gene. The second method is based on the mean methylation beta value of each CpGsite in a gene, and I take the mean value of it as the representative. The third method is based on the pvalue of each CpGsite in a gene comparing between cases and controls. And the

CpGsite with the minimal P-value in a gene is taken as the representative. Based on these three methods, I started to do the further analysis to test which of the three mothods is the best for constructing the panel.
